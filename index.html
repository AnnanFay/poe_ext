Poe Helper

Path of Exile Inventory Management and Crafting Helper (Chrome Plugin)

Download this project as a .zip file Download this project as a tar.gz file

What does it do?

An extension to the chrome web browser that allows you to find crafting recipes and helps to organise your inventory. Initially developed by HouseAbsolute (@jaguilar) then updated by TJJ (@TehJumpingJawa). I have taken the sourcecode and mutilated it beyond recognition.

Installing the Extension

  1. Download the poe_ext.crx file.
  2. Find wherever you saved the file.
  3. In chrome type chrome://extensions into the address bar.
  4. Drag and drop the file from step one into this page.

Using the Extension

  1. Log in to pathofexile.com with "Remember Me" checked.
  2. Open a new page in chrome (ctrl-t or cmd-t on mac).
  3. Select PoE Helper from the Apps (it should have a PoE logo).
  4. Follow the instructions that haven't been written yet!
